# ragkit: ragkit is a Go package to help index and retrieve documents for a RAG server

![ragkit_logo](_asset/ragkit_logo_256.webp)

Install:
```sh
go get github.com/suapapa/go_ragkit
```

Usage:
```go
import ragkit "github.com/suapapa/go_ragkit"
```
